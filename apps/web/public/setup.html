Setup / any agent, five minutes

Wire in memory.

The CLI does everything automatically. If you prefer to configure MCP by hand, pick your IDE below and copy the generated block.

Recommended

One command, zero config

Detects your email from git, creates the agent, installs the MCP server for every editor you have, saves the config. Thirty seconds.

npx @clude/sdk setup
/ 01

Pick your IDE

Clude talks to any MCP-compatible editor. Click yours to get the right config file path:

Config file: .mcp.json in your project root
/ 02

Add your API key optional

Leave blank to run in local mode (SQLite at ~/.clude/brain.db, fully offline). Paste a key from register to sync memories across devices.

/ 03

Copy the config

Paste this block into the config file shown in step 1. If the file already has mcpServers, merge the clude key into the existing object.

After pasting, restart your IDE. Verify with npx @clude/sdk doctor or just ask your agent to store a memory.
