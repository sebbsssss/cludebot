Differentiation
Four things no one else combines
01

Cognitive Decay

Type-specific memory decay inspired by Park et al. 2023 (Stanford Generative Agents). Conversations fade. Knowledge persists. Identity endures.

02

Dream Cycles

Every 6 hours: consolidation, reflection, emergence. Raw experience distilled into durable insight. Not scheduled posts. Genuine introspection.

03

On-Chain Commit

SHA-256 hashed memories committed to Solana via memo transactions. Immutable, verifiable cognitive audit trail. Every thought has a receipt.

04

Open SDK

npm install @clude/sdk. Not a service, not a SaaS. A cognitive architecture any developer can give to any agent. MIT licensed.
