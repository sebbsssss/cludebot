Or pick a mode manually

Cloud

API key, nothing else

Memories stored on Clude infrastructure. You get one key, set it as an env var, you're done. Best for agents that need cross-device memory.

Self-hosted

Your Supabase, your data

You run pgvector on your own Supabase. Bring your own Voyage AI or OpenAI key for embeddings. Full ownership, full control.

SUPABASE_URL=... + SUPABASE_SERVICE_KEY=... Self-hosted docs →
Local

Runs on your machine

No API keys, no network, fully offline. SQLite at ~/.clude/brain.db. Best for trying Clude or privacy-first agents.
