The Problem

AI agents start from zero. Every. Single. Time.

It forgets you

Every session starts from zero. Your AI has no idea what you talked about yesterday.

It never learns

It stores flat facts but doesn't consolidate, reflect, or build understanding over time.

It's not yours

Your conversation history is locked in one provider. No portability, no sovereignty.

How It Works

Store. Recall. Dream.

1

Store

Memories are stored with semantic embeddings, importance scores, and emotional valence.

2

Recall

Hybrid search — vector + keyword + graph — retrieves relevant memories in under 200ms.

3

Dream

Periodic dream cycles consolidate memories, resolve contradictions, and surface emergent insights.

No manual curation. The memory system runs autonomously.

Get Started

Up and running in 5 minutes.

# Install and configure

$ npx clude init

// Add to your Claude Code / Cursor MCP config:

{
  "mcpServers": {
    "clude-memory": {
      "command": "npx",
      "args": ["clude", "mcp"]
    }
  }
}

No API keys needed. Works offline.

# Install the SDK

$ npm install clude-bot

import { Cortex } from 'clude-bot';

const cortex = new Cortex({ owner: 'my-agent' });

// Store a memory
await cortex.store('User prefers dark mode');

// Recall relevant memories
const memories = await cortex.recall('user preferences');
