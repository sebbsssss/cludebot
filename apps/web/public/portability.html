Export formats

Five output shapes, one npx @clude/sdk export command.

chatgpt

Custom Instructions format. Paste into ChatGPT's memory or system prompt.

gemini

Structured for Gemini Gems. Optimised for Google's context format.

md

Markdown file. Works with Claude Projects, Cursor, or any AI that accepts files.

json

Full structured export. All metadata, scores, timestamps. For developers.

clipboard

Copies directly to clipboard. Quick paste into any conversation.

/ 02

By platform

ChatGPT

Export + Import

Export to ChatGPT

  1. Run the export command: 
    npx @clude/sdk export --format chatgpt
  2. Open ChatGPT → Settings → Personalization → Custom Instructions.
  3. Paste the output into "What would you like ChatGPT to know about you?".
  4. Save. ChatGPT now has your Clude memories as context.
Tip. The ChatGPT format automatically prioritises your most important memories and stays under the character limit. It groups them by type: facts, behaviours, and recent context.

Import from ChatGPT

  1. In ChatGPT, go to Settings → Data Controls → Export Data.
  2. Download the ZIP (contains conversations.json).
  3. Run: 
    npx @clude/sdk import chatgpt-export.zip

Clude extracts key facts, preferences, and context from your ChatGPT conversations and stores them as structured memories.

Claude

Export + MCP

Option A: Project Knowledge

  1. Run: 
    npx @clude/sdk export --format md
  2. Open Claude → Create a Project.
  3. Upload the exported .md file as Project Knowledge.
  4. Claude now has your full memory context inside every project conversation.

Option B: MCP integration recommended

Connect Clude directly to Claude Code or Claude Desktop as a live memory layer.

  1. Run: 
    npx @clude/sdk mcp-install
  2. Restart Claude.
  3. Claude can now store_memory and recall_memories directly. No manual export needed.
MCP gives Claude live access to your memory. It stores new memories during conversations and recalls relevant ones automatically. The deepest integration available.

Gemini

Export

Export to Gemini Gems

  1. Run: 
    npx @clude/sdk export --format gemini
  2. Open Gemini → Gems → Create a Gem.
  3. Paste the output into the Gem's instructions.
  4. Save. Your Gem now carries your memory context.
Tip. Gemini Gems have a ~1500-word instruction limit. The export automatically truncates to fit.

Cursor, VS Code, Windsurf

MCP
  1. Run: 
    npx @clude/sdk mcp-install
  2. Select your IDE when prompted (Cursor, VS Code, or Windsurf).
  3. Restart your IDE.
  4. Your coding assistant now has persistent memory across sessions.

Works with any IDE that supports MCP (Model Context Protocol).

Any AI via Markdown

Universal 
npx @clude/sdk export --format md

Produces a structured markdown file with all memories grouped by type. Works with:

  • Claude Projects (upload as knowledge)
  • ChatGPT (upload as file)
  • Obsidian, Notion, Logseq (paste directly)
  • Any AI that accepts file uploads or system prompts
