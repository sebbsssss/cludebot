🟢 ChatGPT Export + Import

Export to ChatGPT

  1. Run the export command: 
    npx @clude/sdk export --format chatgpt
  2. Open ChatGPT → Settings → Personalization → Custom Instructions
  3. Paste the output into the "What would you like ChatGPT to know about you?" field
  4. Save. ChatGPT now has your Clude memories as context.
Tip: The ChatGPT format automatically prioritizes your most important memories and stays under the character limit. It groups them by type: facts, behaviors, and recent context.

Import from ChatGPT

  1. In ChatGPT, go to Settings → Data Controls → Export Data
  2. Download the ZIP file (contains conversations.json)
  3. Run: 
    npx @clude/sdk import chatgpt-export.zip

Clude will extract key facts, preferences, and context from your ChatGPT conversations and store them as structured memories.

🟠 Claude Export + MCP

Option A: Export as Project Knowledge

  1. Run: 
    npx @clude/sdk export --format md
  2. Open Claude → Create a Project
  3. Upload the exported .md file as Project Knowledge
  4. Claude now has your full memory context in every conversation within that project.

Option B: MCP Integration (recommended)

Connect Clude directly to Claude Desktop or Claude Code as a persistent memory layer.

  1. Run: 
    npx @clude/sdk mcp-install
  2. Restart Claude Desktop
  3. Claude can now store_memory and recall_memories directly. No manual export needed.
MCP gives Claude live access to your memory. It can store new memories during conversations and recall relevant ones automatically. This is the deepest integration available.

🔵 Gemini Export

Export to Gemini Gems

  1. Run: 
    npx @clude/sdk export --format gemini
  2. Open Gemini → Gems → Create a Gem
  3. Paste the output into the Gem's instructions
  4. Save. Your Gem now carries your memory context.
Tip: Gemini Gems have a ~1500 word instruction limit. The export automatically truncates to fit.
⚡ Cursor / VS Code / Windsurf MCP

MCP Integration

  1. Run: 
    npx @clude/sdk mcp-install
  2. Select your IDE when prompted (Cursor, VS Code, or Windsurf)
  3. Restart your IDE
  4. Your coding assistant now has persistent memory across sessions.

Works with any IDE that supports MCP (Model Context Protocol).

📄 Any AI (Markdown) Universal

Export as Markdown

npx @clude/sdk export --format md

Produces a structured Markdown file with all memories grouped by type. Works with:

  • Claude Projects (upload as knowledge)
  • ChatGPT (upload as file)
  • Obsidian / Notion (paste directly)
  • Any AI that accepts file uploads or system prompts

Advanced Options

Filter by memory type

npx @clude/sdk export --format chatgpt --types semantic,procedural

Export as JSON (full data)

npx @clude/sdk export --format json

Includes all metadata: importance scores, decay factors, timestamps, tags, concepts, access counts. Use this for backups or building custom integrations.

Quick copy to clipboard

npx @clude/sdk export --format clipboard

Copies formatted memories directly to your clipboard. Paste anywhere.
