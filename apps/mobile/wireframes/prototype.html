Can you explain how the memory decay system works in the cortex?
2:14 PM
The memory decay system uses type-specific rates to model natural forgetting. Each memory type decays at a different rate:

Episodic memories decay at 7% per day — event-based, fade fastest.
Semantic knowledge decays at 2% per day — facts are more stable.
Procedural patterns at 3% per day — learned behaviors persist well.
Self-model insights at just 1% per day — identity is most stable.

The decay factor is a float between 0 and 1. Dream cycles can strengthen or consolidate memories, effectively resetting their decay.

How does the dream cycle consolidation work exactly?
2:15 PM
The dream cycle runs four phases:

1. Consolidation — clusters related memories and strengthens their bonds
2. Compaction — merges near-duplicate memories into summaries
3. Reflection — generates insights about patterns across memories
4. Emergence — produces novel connections between distant concepts

Each phase produces its own memory artifacts. The consolidation phase is the most impactful for decay — it resets the decay factor of strengthened memories.
