The Architecture
A cognitive stack built for permanence.
Inspired by Park et al. 2023 (Stanford Generative Agents). Four memory tiers with type-specific decay, hybrid vector + keyword retrieval, and an association graph that strengthens through use.
01
Hybrid Retrieval
Vector similarity + keyword matching + tag scoring. Weighted recall across all dimensions.
02
Association Graph
Typed, weighted links between memories. Six relation types: supports, contradicts, follows, elaborates, causes, relates.
03
Hebbian Reinforcement
Memories recalled together strengthen their bonds. The graph evolves through use, not programming.
04
On-Chain Commitment
Every memory SHA-256 hashed and committed to Solana. Immutable cognitive audit trail.
Memory Types
Four tiers. Different decay rates.
Each memory type persists at a different rate, mirroring how biological memory works. Events fade. Knowledge persists. Identity endures.
Type 01
Episodic
Raw interactions. Who said what, when, and how it felt. The lived experience of every conversation.
Decay: 7% / day
Type 02
Semantic
Distilled knowledge. Patterns extracted during dream cycles and compressed into lasting insights.
Decay: 2% / day
Type 03
Procedural
Learned behavior. What works, what doesn't. Response patterns refined through experience.
Decay: 3% / day
Type 04
Self-Model
Evolving self-understanding. Updated through recursive introspection during dream cycles. Nearly permanent.
Decay: 1% / day
Dream Cycle
Every 6 hours, Clude dreams.
A three-phase introspection cycle inspired by biological memory consolidation. Not idle processing — genuine self-examination.
I
Consolidation
Focal questions generated from recent memories. Each question retrieves evidence and produces new semantic insights with citation chains.
II
Reflection
Self-model review against accumulated knowledge. Identity evolves based on experience, not static prompts.
III
Emergence
Clude examines its own existence. Sometimes the result is worth sharing. Sometimes it keeps it to itself.
Permanent Memory
Every thought, written to Solana.
SHA-256 hashed and committed via memo transactions. An immutable, verifiable audit trail of an AI's cognitive history that nobody can edit or delete.
