Portability

Your memory, everywhere

Clude stores your AI memory in one place — then takes it wherever you go. Claude, ChatGPT, Gemini, or your own models. Switch freely. Never start over.

Claude Desktop & Code

Native MCP integration — one command, full memory access

Setup

Install the MCP server

Claude Desktop and Claude Code connect directly to your memory via MCP.

npx clude-bot mcp-install
Result

Claude remembers everything

Your preferences, project context, and past conversations are automatically available. No copy-pasting, no setup files.

ChatGPT

Export your memory as a system prompt — paste into Custom Instructions

Clude → Export → ChatGPT
Step 1

Export for ChatGPT

Generates a system prompt with your key facts, learned behaviors, and recent context.

npx clude-bot export --format chatgpt
Step 2

Paste into ChatGPT

Go to Settings → Personalization → Custom Instructions and paste the output. ChatGPT will now have your memory context.

Or: Clipboard

One-step copy

npx clude-bot export --format clipboard

Copies directly to your clipboard. Just paste.

Import

Bring ChatGPT memories into Clude

Download your data from ChatGPT (Settings → Data Controls → Export Data), then import the ZIP.

npx clude-bot import chatgpt-export.zip

We extract preferences, corrections, and key facts from your conversations.

Gemini

Export your memory for Gemini Gems custom instructions

Clude → Export → Gemini Gem
Step 1

Export for Gemini

npx clude-bot export --format gemini
Step 2

Create a Gem

Go to Gemini → Gems → Create and paste the exported instructions. Your Gem now has full context about you.

Any AI

REST API + SDK for custom integrations

# Store a memory
curl -X POST https://cluude.ai/api/cortex/memory \
  -H "Authorization: Bearer YOUR_KEY" \
  -d '{"content": "User prefers TypeScript", "type": "semantic"}'

# Recall relevant memories
curl -X POST https://cluude.ai/api/cortex/recall \
  -H "Authorization: Bearer YOUR_KEY" \
  -d '{"query": "programming preferences"}'
# Python SDK
from clude import Cortex

brain = Cortex(api_key="YOUR_KEY")
brain.store("User prefers dark mode", type="semantic")
memories = brain.recall("UI preferences")
