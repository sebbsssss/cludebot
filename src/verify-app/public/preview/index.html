The Problem

AI agents are amnesiacs.
Every restart, every session — gone.

Context Windows

Limited tokens. Expensive to maintain. Ephemeral by design.

Vector Databases

Private silos. Unverifiable retrieval. Single point of failure.

File Storage

Manual management. No provenance. Lost on disk failure.

The Solution

Three-Layer Memory

Local speed when you need it. Chain permanence when it matters. Automatic promotion based on importance.

L3 — WORKING
In-Context Memory
Active task context. Managed by LLM.
~0
ms latency
L2 — LOCAL
SQLite + FTS5
Instant recall. Semantic search. Private.
<1
ms latency
L1 — CHAIN
Solana
Permanent. Verifiable. Shareable.
400
ms finality
Cognitive Architecture

Four Memory Types

Inspired by human cognition. Each type serves a distinct purpose.

EPISODIC

Significant interactions and events. The "what happened" of agent memory.

"User asked about dark mode on Feb 14"

SEMANTIC

Distilled knowledge and facts. Patterns learned over time.

"Parallel processing improves latency"

PROCEDURAL

How to do things. Behavioral patterns and workflows.

"Always run tests before deploy"

SELF-MODEL

Agent's understanding of itself. Capabilities and limits.

"I tend to be verbose technically"
